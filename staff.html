📹 Live Meetings & Class Recording

How it works: 1) Fill the meeting title below and press Create Meeting — the site makes a unique code.   2) Press Publish Changes at the bottom so the code becomes active.   3) Send the code to your students.   4) When it's time, press ▶ Start & Host to open the room yourself first — students then join with the code. You can record the class and turn on live translated captions inside the room.

Just want to record a lesson (no students)? Create a meeting named e.g. "Recording — Tafsir 1", publish, press ▶ Start & Host, use the ⏺ Record button in the room, then upload the saved file in the "Add a Paid Video" panel above.
